🔄 מדריך סנכרון בין מכשירים

האפליקציה שומרת את המתכונים בכל מכשיר בנפרד (באחסון של הדפדפן). כדי שהאוסף שלך יהיה זהה בפולד, במחשב ובשעון, צריך לבצע סנכרון ידני באמצעות קובץ גיבוי.

📥 כדי לסנכרן ממכשיר אחד לאחר:

  1. במכשיר המקור (זה שיש בו את הגרסה החדשה ביותר): פתחי את התפריט ⋮ ולחצי "גיבוי - הורדת קובץ". ייווצר קובץ JSON עם תאריך היום.
  2. העבירי את הקובץ למכשיר היעד - דרך וואטסאפ לעצמך, מייל, OneDrive, או כל דרך אחרת.
  3. במכשיר היעד: פתחי את האפליקציה, תפריט ⋮ ← "שחזור מקובץ גיבוי", בחרי את הקובץ.
  4. הנתונים יוחלפו במלואם. ⚠️ הגיבוי מחליף את הקיים - אז ודאי שאת מסנכרנת מהמכשיר המעודכן.
💡 טיפ: הגדירי לעצמך הרגל - כל פעם שמוסיפה מתכון חדש או עורכת, מבצעת גיבוי קצר באותו יום. ככה תמיד יש גיבוי עדכני.

📲 בפולד שלך (Galaxy Z Fold 7):

  • פתחי את האפליקציה דרך הקישור או דרך הקיצור במסך הבית
  • הקובץ יורד לתיקיית Download או הורדות
  • בשחזור - בחרי את הקובץ דרך מנהל הקבצים

💻 במחשב הלנובו:

  • הקובץ יורד לתיקיית Downloads
  • אם את משתמשת ב-OneDrive, את יכולה לשמור את הגיבוי שם - והוא יתסנכרן אוטומטית למכשירים אחרים שמחוברים לאותו OneDrive

⌚ בשעון Galaxy Watch 7:

  • אין תמיכה מלאה בקבצים מקומיים בשעון
  • עדיף לפתוח את האפליקציה בפולד או במחשב לעריכה ולסנכרון
  • בשעון - לצפייה במתכון בלבד
🔔 תזכורת אוטומטית: האפליקציה תזכיר לך לבצע גיבוי אם עברו 7 ימים מאז הגיבוי האחרון.
